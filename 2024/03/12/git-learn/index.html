远程协作的一些流程

1、首先试图用 git push origin推送自己的修改

2、在某些情况下，因为远程分支比本地更新，需要先用git pull 试图合并

3、如果合并有冲突，则解决冲突，并在本地提交

4、没有冲突或者解决掉冲突后，再用git push origin推送就能成功
